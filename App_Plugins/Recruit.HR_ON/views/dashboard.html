Welcome to HR-ON in Umbraco

When installed this package extends Umbraco with HR-ON (Recruit) functionality.

This functionality allows Umbraco pages to show/display your list of jobpostings from Recruit.

This package includes the following: Document types, Templates and Partial views.

1. Partial views

The 'HR-ON' partial view includes the code to insert the HTML and Scripts.

2. Templates

The 'HR-ON' template is just a 'wrapper' around the partial view to make usage easier.

3. Document types

The 'HR-ON' document type is a default page for showing/displaying the jobposting list. It simply wraps the 'HR-ON' template and defines the inputs/variables needed for the partial view. These inputs being: 'scriptName' and 'containerName'.

(The values for 'scriptName' and 'containerName' can be found in the HR'ON onboarding document.)

Customization:

The package provides default code for getting a page with jobpostings up and running. However you can change these files directly to make changes, or you can make new templates and partial views based on, or inluding the default onces. You can take a look in the 'HR-ON' template and partial view to see how this can be done.

You can also make your own document type if needed but it must contain the 'scriptName' and 'containerName' variable for the jobposting list to work.
